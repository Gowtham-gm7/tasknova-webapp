# 🚀 TaskNova – Smart Productivity & Task Management Platform

TaskNova is a modern, responsive, and feature-rich **Task Management & Productivity Web Application** built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. It helps users organize daily tasks, boost productivity, and manage workflows efficiently with a beautiful and intuitive user interface.

🌐 **Live Demo:** https://tasknova-webapp.netlify.app

# ✨ Features

## 📋 Smart Task Management

- Create, Edit, Delete & Duplicate Tasks
- Organize tasks with Categories
- Set Task Priority (Low, Medium, High)
- Due Date Management
- Search & Filter Tasks
- Mark Tasks as Completed
- Unlimited Subtasks
- Live Progress Indicator
